# GOLANG Basics

A collection of Go programming tutorials covering fundamental concepts.

## Topics Covered

| # | Topic | Description |
|---|-------|-------------|
| 1 | [Hello GO](./1_Hello_GO) | First Go program, `main` package, `fmt.Println()` |
| 2 | [Variables](./2_Variables) | Variable declaration, types, constants, zero values |
| 3 | [Printing & Formatting](./3_PrintingAndFormatingString) | `Printf`, format verbs (`%s`, `%d`, `%v`), string formatting |
| 4 | [Arrays & Slices](./4_ArraysAndSlice) | Fixed-size arrays, dynamic slices, `append()`, `len()`, `cap()` |

## Getting Started

### Prerequisites
- [Go](https://go.dev/dl/) installed (version 1.21+)

### Running Examples
```bash
cd 1_Hello_GO
go run main.go
```
